Language learning app

Create a quiz from a book or any subject of interest e.g food greetings or just lesson 1. The words can be looked up in online dictionary (yandex) or entered manually. Listen to the pronunciation of the words (from yandex and voicerss.org)

To create a quiz

  1. Enter a name in the Tab "Home"
  2. Select Language pair e.g Swedish-Russian
  3. Press "New Quiz"
  4. Select tab "Edit"
  5. Enter a word e.g hund (dog) in this case pres button "sv-ru"
  6. Select if you like a synonym by pressing it
  7. Press "Add"
  8. Repeat until you have a convenient number of questions for a WordQuiz.
  9. To run the Quiz press "Quiz"
  10. Go a head and swipe left , if you dont know the answer or right if you are confident until you pass the exam!
  11. If you find some thing wrong go back to "Edit", the question should be selected, press the edit symbol and fix the error. You can also include extra information e.g word class

To Download an available Quiz

  1. Press "Download"
  2. Scroll around, a description off the quiz i shown at the top
  3. Select a quiz with prefered language combination
  4. Press "Download"

To Upload a Quiz

  1. Press "Upload"
  2. Add a short description
  3. Enter a word, this is needed if you want to remove the Quiz
  4. Press "Upload"

To Edit a Quiz

  1. Press "Edit" tab
  2. Select the question with the edit symbol
  3. Enter the new information
  4. You can fill the paste buffer by first pressing and hold on the word and then press the copy symbol. Then you can search for more information in a browser

To Delete an uploaded Quiz

  1. Press "Download"
  2. Select the quiz to remove
  3. Press "Remove"
  4. Enter the password entered when it was uploaded
